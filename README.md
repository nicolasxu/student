## How to run the code
1. Make sure [Node.js](http://nodejs.org/) and NPM package manager are installed
2. Install all dependencies:
  ```
  $ npm install
  $
  ```
3. Run the dev server
  ```
  $ npm run dev
  $
  ```
4. Open the url in browser. The URL is shown in above dev server, normally
   ```
   http://127.0.0.1:5173/#/
   ```
